Introduction

Welcome to our enhanced documentation template. This version includes the ability to load external HTML files in the main content area while maintaining all the previous functionality.

Notice the two external links in the navigation marked with an arrow icon (↗). These will load separate HTML files in this content area.

Getting Started

To use this template:

  1. Create your external HTML files (like external1.html and external2.html)
  2. Add navigation links with the data-external attribute
  3. Ensure your external files have the same styling or include the same CSS

Features Overview

New features in this version:

  • Load external HTML files in the main content area
  • Visual indicator for external links (arrow icon)
  • Smooth transition between internal and external content
  • Maintained all previous functionality (search, highlighting, etc.)

Configuration Options

For the external content loading:

  • Add data-external attribute to external links
  • Use the external-link class for visual styling
  • Ensure external files are in the same directory or update paths
